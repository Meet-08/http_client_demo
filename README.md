# Spring Boot HTTP Client Demo

This project demonstrates different ways to consume REST APIs in Spring Boot, specifically targeting the JSONPlaceholder API.

## Features

- **Old Implementation**: Using `RestClient` (labeled as `RestTemplateOld`).
- **New Implementation**: Using Spring boot 4 Declarative HTTP Client with `@HttpExchange` (labeled as `RestTemplateNew`).

## Requirements

- Java 17 or later
- Gradle

## How to Run

Run the application using the Gradle wrapper:

```bash
./gradlew bootRun
```
